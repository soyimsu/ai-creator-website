# PRD：AI内容创作者个人网站

## 一、项目概述

| 项目 | 说明 |
|------|------|
| 项目名称 | AI内容创作者个人品牌网站 |
| 目标用户 | 对AI感兴趣的职场人 |
| 核心目标 | 建立个人品牌，展示专业内容，引流至小红书 |
| 语言 | 中英双语（同一内容可切换语言） |
| 域名 | 待定 |

---

## 二、网站结构

```
首页 (Home)
├── 教程文章 (Tutorials)
│   ├── 提示词 (Prompts)
│   ├── Skills
│   └── AI+投资 (AI + Investment)
├── 资源推荐 (Resources)
└── 关于我 (About)
```

---

## 三、页面详细需求

### 3.1 首页 (Home)

| 模块 | 说明 |
|------|------|
| 顶部导航栏 | Logo/网站名称、导航链接（教程文章、资源推荐、关于我）、中英文切换按钮 |
| Banner区域 | 大幅Banner，包含个人标语（Tagline）和一句话简介，配插画或个人形象 |
| 最新文章 | 展示最近3-6篇教程文章卡片（封面图、标题、分类标签、发布日期） |
| 资源推荐精选 | 精选3-6个资源卡片（资源名称、一句话描述、链接） |
| 页脚 | 小红书主页链接、版权信息 |

### 3.2 教程文章 (Tutorials)

| 模块 | 说明 |
|------|------|
| 文章列表页 | 按主题分类筛选（提示词 / Skills / AI+投资），卡片式展示 |
| 文章详情页 | 标题、发布日期、所属分类、正文内容、中英文版本切换 |
| 更新频率 | 每周2-3篇 |

### 3.3 资源推荐 (Resources)

| 模块 | 说明 |
|------|------|
| 展示形式 | 资源列表/卡片墙 |
| 每个资源包含 | 资源名称、一句话描述、外部链接 |

### 3.4 关于我 (About)

| 模块 | 说明 |
|------|------|
| 个人照片/头像 | 展示个人形象 |
| 个人简介 | 一段自我介绍 |
| 职业经历 | 时间线或简要列表形式 |
| AI创作理念 | 一段关于为什么做AI内容的阐述 |
| 小红书引流 | 突出展示小红书主页链接，引导用户关注 |

## 四、功能需求

### 4.1 中英双语切换

- 全站支持中文/英文切换（导航栏设置切换按钮）
- 同一篇文章对应中英两个版本，用户可自由切换
- 切换语言时页面结构保持一致，仅内容语言变化
- 默认语言：中文

### 4.2 后台内容管理 (CMS)

- 需要一个易用的后台管理系统（无需编码）
- 支持新建/编辑/删除教程文章
- 支持为文章设置分类标签
- 支持为同一篇文章录入中英两个版本
- 支持管理资源推荐列表（增删改）
- 支持上传图片

### 4.3 不需要的功能

- 不需要评论系统
- 不需要用户注册/登录
- 不需要Newsletter/邮件订阅
- 不需要搜索功能（初期）

## 五、设计风格

| 项目 | 说明 |
|------|------|
| 整体风格 | 温暖亲和、明亮配色、插画风 |
| 配色方向 | 明亮暖色调（如柔和的橙色、黄色、浅粉色等，具体配色待定） |
| 字体 | 中文：无衬线体（如思源黑体）；英文：现代无衬线体（如Inter） |
| 插画元素 | 可在Banner、空状态、装饰区域使用插画，增强亲和力 |
| 响应式 | 适配桌面端和移动端 |

## 六、技术方案建议

由于不会代码且需要频繁更新内容，推荐以下低代码方案：

| 方案 | 说明 | 适合度 |
|------|------|--------|
| **Framer** | 设计自由度高，支持CMS，适合品牌展示站 | ★★★★★ |
| **Webflow + CMS** | 专业建站工具，CMS功能强大，支持多语言 | ★★★★★ |
| **WordPress + Elementor** | 生态成熟，插件丰富，多语言方案完善（WPML插件） | ★★★★☆ |
| **Notion + Super.so** | 最简单，但设计自由度和多语言支持有限 | ★★★☆☆ |

> **推荐首选：Framer 或 Webflow**——设计自由度高、自带CMS、对多语言友好、无需代码。
